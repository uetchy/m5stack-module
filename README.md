# Custom Settings

## Enclosure Height

```openscad
enclosure_height = 6.8;
```

Place a number for height of the enclosure you want to build.

![](assets/s3.png)

![](assets/s4.png)

## Bottom Plate

```openscad
build_bottom_plate = true;
```

Set `true` or `false ` for building bottom plate for a enclosure.

![](assets/s5.png)
