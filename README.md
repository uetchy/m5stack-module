# Custom Settings

## Enclosure Height

```openscad
enclosure_height = 6.8;
```

Place a number for height of an enclosure you want to build.

![](assets/s3.png)

![](assets/s4.png)

## Bottom Plate

```openscad
build_bottom_plate = true;
```

Set `true` to build a bottom plate for an enclosure.

![](assets/s5.png)
